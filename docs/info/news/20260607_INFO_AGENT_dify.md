# Dify — AIエージェント／LLMアプリを作るオープンソース開発基盤

> 作成日: 2026-06-07 / STATUS: INFO / TOPIC: AGENT
> ※ 個別指定で作成した紹介記事です。

## 概要

[langgenius/dify](https://github.com/langgenius/dify) は、「**本番運用に耐えるエージェント型ワークフロー開発プラットフォーム**」を掲げるオープンソースのLLMアプリ開発基盤です。ブラウザ上の画面（ローコード）で、チャットボットやAIエージェント、業務自動化フローを組み立て、そのまま API として公開できます。

- **スター数**: 約 144,235 ⭐
- **フォーク数**: 約 22,701
- **主要言語**: TypeScript（バックエンドは Python も併用）
- **作成**: 2023-04-12 / 最終更新: 2026-06-07（非常に活発）
- **ライセンス**: 独自表記（NOASSERTION。商用利用条件があるため LICENSE 要確認）

> 補足: **AIエージェント** とは、目標を与えると自分で手順を考え、ツールを呼び出しながらタスクを進めるAIのこと。**ローコード** は、コードをほぼ書かずに画面操作で開発する手法です。

## なぜ注目なのか

1. **作って公開までが速い** — プロンプト設計・データ接続・公開を1つの画面で完結。
2. **RAG 標準対応** — 自社ドキュメントを読み込ませて回答精度を上げる仕組み（検索拡張生成）を内蔵。
3. **多数のLLMに対応** — OpenAI・Gemini・各種オープンモデルなどを差し替えて使える。
4. **エージェント／ワークフロー指向** — 単発の応答だけでなく、複数ステップの自動処理を組める。

## 主な使いどころ

- 社内ナレッジを参照するチャットボット構築
- 問い合わせ対応・文書要約などの業務自動化
- LLMアプリのプロトタイピングと本番運用
