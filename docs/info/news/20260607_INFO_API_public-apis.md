# public-apis — 無料で使える API を集めた巨大カタログ

> 作成日: 2026-06-07 / STATUS: INFO / TOPIC: API
> ※ 個別指定で作成した紹介記事です。

## 概要

[public-apis/public-apis](https://github.com/public-apis/public-apis) は、「**無料で使える公開 API（Application Programming Interface）**」を分野別に集めた一覧リポジトリです。天気・金融・地図・機械学習・ゲームなど、カテゴリごとに「どんな API があるか／認証は必要か／HTTPS 対応か」が表形式でまとまっています。

- **スター数**: 約 439,905 ⭐（GitHub 最上位クラス）
- **フォーク数**: 約 48,210
- **主要言語**: Python（リンク検証などの補助ツール）
- **作成**: 2016-03-20 / 最終更新: 2026-06-07（現在も活発に更新）
- **ライセンス**: MIT（商用含め自由に利用しやすい寛容型ライセンス）

> 補足: **API** とは、あるサービスの機能やデータを、プログラムから呼び出して使うための「窓口」です。例えば天気 API を呼べば、自分のアプリに最新の天気を組み込めます。

## なぜ人気なのか

1. **個人開発・学習の起点になる** — 「何か作りたいがデータが無い」時に、すぐ使える題材が見つかる。
2. **網羅性と分類のしやすさ** — 数百の API がカテゴリ分けされ、認証要否や CORS 対応の有無も一覧化。
3. **メンテナンスが継続** — 古くなった API の整理やリンク切れチェックが続いている。

## 主な使いどころ

- ハッカソン・ポートフォリオ用のサンプルアプリのデータ源探し
- API 連携（外部サービス呼び出し）の練習
- 各分野でどんな公開データ・サービスが存在するかのリサーチ

## 対象読者

- これからアプリを作りたい初学者〜中級の開発者
- 新しいサイドプロジェクトのネタやデータ源を探している人
