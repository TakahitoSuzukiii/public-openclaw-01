## 3. X（旧Twitter）・SNS の活用

特定アカウントは入れ替わりが激しいため、ハンドル直指定よりも**検索クエリでの巡回**を推奨：

- `テンプレート 配布 パワポ` / `#パワポ #デザイン`
- `パワーポイント 無料 配布`（pptx 配布ポストが見つかる）
- `Excel テンプレート 配布` / `#エクセル時短`
- デザイナー/資料作成系の発信者を1〜2人フォローしておくと配布告知が流れてくる。

※ X の配布物は**個人配布**が多く、ライセンスが曖昧なものもあるため商用利用は要注意。

## 4. 「ワンクリックアクセス」で管理する方法

発見したサイトを日々ワンクリックで開けるよう、リンク集として管理する案：

1. **自作のリンク集ページ**（ローカル HTML / 社内ツール等）にカード化し、ボタン1つで各サイトを新規タブで開く。
2. ブラウザのブックマークバー／フォルダにまとめてインポート（URL 一覧から `.html` ブックマークを生成可能）。
3. Notion / スプレッドシートにリンク集として転記（共有・スマホ閲覧向き）。

## 5. LLM からの Excel/PPT 生成・編集について（補足）

「LLM／LLMチャットから新規作成・編集・更新」したい場合、テンプレートとは別に生成手段が要ります。候補：

- **python-pptx / openpyxl**（Python ライブラリ）: テンプレ .pptx/.xlsx をベースに、プレースホルダを差し替えて量産する自動化に向く。
- **Microsoft Graph API**: Microsoft 365 上のファイルを API 操作（要 M365 / 認証）。
- 上記テンプレを「ひな型」として読み込み → 内容を流し込む、という構成が現実的。
